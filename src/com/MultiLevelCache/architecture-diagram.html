Design Patterns & SOLID Principles Applied

Strategy Pattern

EvictionPolicy interface → swap LRU for LFU, FIFO, TTL without touching CacheLevelImpl

Factory Pattern

CacheLevelFactory centralizes level creation with sensible defaults (LRU)

Dependency Inversion

Service depends on CacheLevel, StatsTracker, EvictionPolicy interfaces — never concrete classes

Single Responsibility

Stats tracking extracted to StatsTracker. Cache logic stays in service. Storage in CacheLevel.

Open/Closed

New eviction, stats, or cache level types plug in via interfaces — no core code changes needed
