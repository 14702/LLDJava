Sequential Message Flow

1
Producer
Creates Message with JSON payload + TTL
→
2
Broker
publish() puts msg into Queue
→
3
BoundedQueue
Stores in circular buffer (lock-guarded)
→
4
Dispatch
take() → skip if expired → fan out to consumers
→
5
Filter
Check dependencies (latch) then matches()
→
6
Callback
onMessage() with retry. Latch countdown in finally.
